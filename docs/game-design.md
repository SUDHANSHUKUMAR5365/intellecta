# Intellecta Game Design v1

## Core Concept

Intellecta is a knowledge RPG where users learn, earn rewards, level up, unlock worlds, and compete on leaderboards.

---

# Currencies

## XP

Used for leveling.

Sources:

* Correct Answers
* Daily Challenges
* Boss Battles
* Achievements

---

## Coins

Used for:

* Hints
* 50-50
* Powerups
* Cosmetic Items

---

## Gems

Premium currency.

Earned from:

* Major Achievements
* Boss Battles
* Seasonal Rewards

---

# XP Rewards

Easy Question = 10 XP

Medium Question = 25 XP

Hard Question = 50 XP

UPSC Question = 100 XP

---

# Coin Rewards

Easy Question = 5 Coins

Medium Question = 10 Coins

Hard Question = 20 Coins

UPSC Question = 40 Coins

---

# Levels

Level 1 = 0 XP

Level 2 = 100 XP

Level 3 = 250 XP

Level 4 = 500 XP

Level 5 = 1000 XP

Then continue with scaling progression.
